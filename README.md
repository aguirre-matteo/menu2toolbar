# `menu2toolbar`
This Firefox extension will copy all the contents from the Bookmark Menu to the Toolbar. This is a fix for [this](https://github.com/nix-community/home-manager/pull/7399#issuecomment-3628263735) issue when declaratively configuring Bookmarks in Firefox (or derivatives) with Home-Manager.
